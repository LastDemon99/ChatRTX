“传送门：序曲 (Portal: Prelude)”RTX 版现已推出 - 免费 RTX Remix Mod 支持全景光线追踪、NVIDIA DLSS 3 和 NVIDIA RTX IO

“传送门 (Portal)”是玩家心目中的经典；去年 12 月，我们发布了基于 Valve 获奖游戏重构的游戏“传送门 RTX 版 (Portal with RTX)”，支持全景光线追踪，并由 NVIDIA DLSS 3 提供性能提升，在实时游戏中实现了逼真的路径追踪光照。

“传送门 RTX 版 (Portal with RTX)”使用 NVIDIA RTX Remix 创建而成，NVIDIA RTX Remix 是一款免费的、即将推出的基于 NVIDIA Omniverse 的 MOD 平台，可让 Modder 快速创建和分享经典游戏的 #RTXON Mod，而且每个 Mod 都支持增强材质、全景光线追踪、NVIDIA DLSS 3 和 NVIDIA Reflex。

现在，使用 RTX Remix 重构的“传送门：序曲 (Portal: Prelude)”，是一款高分“传送门 (Portal)” Mod、它是 2008 年 Portal ModDB 获胜者以及 2008 年 Player’s Choice Mod 第三名。由 Nicolas “NykO18” Grevet 等人创建的“传送门：序曲 (Portal: Prelude)”是一款非官方传送门前传，背景设定在 GLaDOS 时代之前，具有全新的极具挑战性的 8-10 小时游戏体验，其中包括 19 个全新实验室、新的高级游戏机制、长达 400 多行的全程 NPC 语音对话，以及更多其他內容。

“传送门：序曲 (Portal: Prelude)”RTX 版今日在 Steam 平台上线！

 
“传送门：序曲 (Portal: Prelude)”RTX 版经过重新制作，采用了全景光线追踪（也称为路径追踪）、DLSS 3、Reflex 和新推出的 NVIDIA RTX IO，为玩家提供更加身临其境的游戏体验。同时，David 和 Valerios “Fearell_Val” Tsoumpas 还提供了数百种全面升级的材质和资产， Jack “Amicus” McDade 和 Colin “SordidSpectacle” Zachariasen 带来了全新配音，Nicolas 对游戏玩法进行了改进。“传送门：序曲 (Portal: Prelude)”RTX 版是体验这款屡获殊荣 Mod 的最佳方式。
“传送门：序曲 (Portal: Prelude)”RTX 版介绍
如果您玩过“传送门 RTX 版 (Portal with RTX)”，您就会了解该游戏的沉浸式全景光线追踪（路径追踪）照明、重构的材质和模型，以及令人难以置信的视觉效果，让您可以预览所有游戏未来的样子。如果您需要回顾或者是 RTX 新手，请在此处查看所有详细信息。

“传送门：序曲 (Portal: Prelude)”RTX 版更加精彩。而且其技术更加先进。

在探索“传送门：序曲 (Portal: Prelude)”的故事过程中，玩家将遇到极具挑战性的先进实验室，其场景更加多样化，借助 RTX 的强大功能，绝对会让您大吃一惊。多层玻璃折射光线，玩家从完全黑暗移动到明亮的位置，粒子分解器等危险以致命的光线照亮世界，利用 RTX 创造难忘的时刻。
200 种新材质和纹理以及 100 种全新模型与“传送门 RTX 版 (Portal with RTX)”的重制素材完美契合， 为实验室和挑战增添了额外的多样性。先进的几何结构在 2008 年突破了 Source 引擎的界限，使光和影的使用更加多样化和美观。对于“传送门：序曲 (Portal: Prelude)”RTX 版，几何体在标准 Half-Life 引擎工具中进行了更新和升级，然后导入并由 RTX Remix 自动支持，从而实现快速迭代和开发。

“传送门：序曲 (Portal: Prelude)”中 NPC 角色的加入带来了新的挑战，需要为 RTX Remix 开发一项令人兴奋的功能：对蒙皮角色的全面支持。现在，在任何采用 GPU 皮肤骨骼动画的游戏中，除了高保真 PBR 纹理之外，Remix 还可以让 Modder 以最低的性能成本为角色添加超过 100 倍的多边形细节。从本质上讲，角色和动画/蒙皮道具可以像 RTX Remix 中的任何对象一样得到增强，从而使未来所有的 RTX Mod 受益。或者换句话说，RTX Remix 正在彻底改变角色替换和升级，就像它彻底改变素材替换一样。


对于“传送门：序曲 (Portal: Prelude)”RTX 版和未来的 NVIDIA RTX Remix Mod，NVIDIA RTX 路径追踪 SDK 进行了增强，提高了降噪器性能和质量，这在较低分辨率下尤其有益，使旧款 GeForce RTX GPU 玩家能够获得更好的体验。

着色器执行重排序 (SER) 动态地将低效的光线追踪运算负载重新组 织的更为高效，进一步提升 GeForce RTX 40 系列的性能。

Opacity Micro-Mesh (OMM) 提升了复杂几何图形的实时性能和内存 压缩效率，进一步加速了 GeForce RTX 40 系列光线追踪性能，并 且随着“传送门：序曲 (Portal: Prelude)”RTX 版中首次推出的全新 CPU 优化，玩家可以获得更快的性能。

当然，NVIDIA DLSS 3 和 NVIDIA Reflex 均已升级到最新版本， 以提升性能、提供更高图像质量和更快响应的游戏体验。


所有为“传送门：序曲 (Portal: Prelude)”RTX 版提供的增强、新增功能和改进都将纳入 NVIDIA RTX Remix，让所有 Modder 受益，并为经典游戏的出色更新提供支持。在不久的将来，NVIDIA 将开始同优选 Modder 共享 RTX Remix 的抢先体验版本，以在今年晚些时候正式发布之前就可以进一步完善创作者工具包和运行时间。请持续关注更新。

NVIDIA RTX IO 首次亮相，加快加载时间并减小文件尺寸
近乎即时的游戏加载和在无尽的开放世界中的无卡顿导航长期以来一直是游戏玩家和开者者的目标。即使 NVMe SSD 具有令人难以置信的性能，这一目标仍然遥不可及。当代游戏引擎已经超越了传统存储 API 的能力；需要新一代的输入/输出（IO）架构。

NVIDIA RTX IO 登场，可实现基于 GPU 的快速加载和游戏资源解压，与硬盘和传统存储 API 相比，可大幅提升 IO 性能。 NVIDIA RTX IO 利用 GPU 解压缩来处理较小的数据包，从而实现更快的加载速度和更低的 CPU 利用率，并允许开发者创建具有大量丰富、细腻世界的新一代游戏。
